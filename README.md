# OpenWeatherMap


An Android Weather App where you can check weather from list of selected cities. It was built with mvvm, dagger-hilt, livedata,coroutines, jetpack libraries...

# Features

1- Kotlin

2- MVVM design pattern

3- DataBinding

4- ViewBinding

5- Coroutine

6- Weather Api

7- LiveData

8- Retrofit 2
